# BSCS 2-1 Student Information and Grade Management System

## Project Summary
This project is a simple console-based Java application designed to manage the student records and grades of **BSCS 2-1** students only.  
It follows the **CRUD (Create, Read, Update, Delete)** approach and applies basic **Object-Oriented Programming (OOP)** concepts.


## Purpose of the System
The purpose of this system is to provide a simple way to **create, read, update, and delete (CRUD)** the student records and grades of BSCS 2-1.


## Programming Language
- Java


## Features
- Add new BSCS 2-1 student records  
- View all stored students  
- Search for a student using ID  
- Update student information and grade  
- Delete a student record


## System Scope
- The system is **exclusive to BSCS 2-1**
- Course, year, and block are fixed as **BSCS 2-1**
- Maximum of **50 students** can be stored
- Console-based (no GUI)
